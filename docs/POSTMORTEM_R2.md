| ガード | 内容 | 場所 |
|---|---|---|
| **通貨×方向 24hキャップ** | 同一通貨・同方向のアラートは24hに最大2件。3件目は自動抑制、2件目に⚠警告 → USDロング×3の再発防止 | `build_static.detect_new_alerts` |

| **MTF に EVゲート** | 緩和版MTFをバックテスト計測し、+EV実証ペアのみ許可 (計測済みになったら whitelist-only) | `ev_whitelist.py` / `backtest.py --emit-whitelist` (手法別マージ対応) |
| **MTF 品質底上げ** | Aランクパターンは重要節目に重なる時のみ (Sは単体可)。節目 +8 加点 | `strategy_mtf.py` |
| **アラート文面** | 「🛡 ロット固定 (口座リスク1%以内)・倍ロット/リベンジ増し玉 禁止」を毎回明記 | `_format_alert` |

## コードでは防げない、あなたが守る3つのルール

1. **ロットは固定**。何連勝しても増やさない。増やすなら「口座残高の1%リスク」の
   計算に従う（10倍などの離散ジャンプは絶対にしない）。
2. **SLは注文と同時にサーバーに置く**。置いた後は動かさない（建値方向のみ可）。
3. **同じ通貨方向のポジションは1つまで**。EURUSD売り＋USDCHF買い＋NZDUSD売りは
   「3ポジション」ではなく「USDロング3倍」だと認識する。
