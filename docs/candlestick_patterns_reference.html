ロウソク足＆チャートパターン集 — 15分足トレード実践

「この形が出たら買い／売り」をローソク足＋チャートパターン（ダブルトップ/ボトム・三角・トレンドの高値安値切り下げ等）まで網羅。信頼度ランク S→C 順。図はコード描画。緑＝買い／赤＝売り。

15分足で使うときの大前提（これを守らないとどの形も機能しない）

  • ① 確定足で判断：形は「:00 / :15 / :30 / :45 にロウソクが確定してから」。形成中の足で飛びつかない。
  • ② 上位足の方向に従う：1時間足・4時間足のトレンド方向と同じ向きのパターンだけ採用（逆張りは難度大）。
  • ③ 「重要な節目」で出た形だけ：水平線・前日高安・キリ番・移動平均線・ボリンジャーバンド端に重なった時のみ信頼度が上がる。何もない所の形は無視。
  • ④ 次の足で確認：反転系は「次の足がパターンの方向に動いて確定」したらエントリー。ダマシを大きく減らせる。
  • ⑤ 資金管理が前提：ロットは ロット計算ツール.html、リスク1〜2%は 資産管理ガイド.md 参照。形は“優位性”を作るだけ。
チャートパターン（ダブルトップ等）の大前提 ＋ 信憑性の考え方
  • ① 確定ブレイクで初めて成立：ネックライン／トレンドラインを15分足の終値で抜けるまでは“未完成”。形が見えただけで入らない。
  • ② リターンムーブで入る：ブレイク後に抜けたラインまで一度戻ってから再度離れる動きが最も安全（ダマシ＝ヒゲだけ抜けを回避）。
  • ③ 値幅目標（メジャードムーブ）：パターンの高さを、ブレイク点から同じ分だけ投影＝利確目安。
  • ④ 上位足の方向と一致で信頼度大。逆方向のパターンは難度が上がる。
  • ⑤ 一般にチャートパターンはローソク足単体より信憑性が高い：多くの足で形成＝多数のトレーダーが同じラインを意識するため。本書は信頼度 S→C 順に並べ、早見表でも上から信憑性が高い。

時間帯（東京時間 ＝ JST）

  • 東京 9:00〜15:00：値動き穏やか。レンジになりやすい＝毛抜き／はらみ／逆張り系が効きやすい。
  • ロンドン 16:00〜（夏時間）：トレンド発生しやすい。包み足・三兵・ブレイク系が機能。
  • NY 21:30〜（指標集中）：最も動く。指標発表の直前直後はパターン無効。発表後に落ち着いてから。

上位足・水平線フィルター手順

  • 1. 4時間足でトレンド（高値安値の切り上げ/切り下げ）と主要水平線を引く。
  • 2. 1時間足で押し目／戻りの位置を確認（＝形が出てほしいゾーン）。
  • 3. 15分足でそのゾーンに来た時だけ、本書の形を待つ。出なければ見送り。
  • 4. 利確はRR1:1.5〜2を基本、または直近の高値安値／上位足の節目。
